DFVA v2 — Prototype Matrix

2×2 design: curriculum adaptiveness × destination exposure. Static prototype — zero code impact.

⚠
This is a design prototype, not a product change.
The live product (evidura.ai) continues to serve v1 bands. This prototype uses real DFVA data to demonstrate the v2 2×2 matrix shape proposed in docs/proud-moseying-taco.md. No report, MCP server, or app code has been modified.
Panel A (X-axis): Destination AI Exposure — mean_title_exposure from JIR destination data (34 of 66 programs have JIR matches). Unweighted mean Felten AIOE of graduate destination occupations, rescaled 0–100. Programs without JIR data (32) appear in the table but not on the chart.

Panel C (Y-axis): Curriculum Adaptiveness — Sum of scored v2 dimensions only: D2 Systems Thinking, D3 Technical Depth, D7 Research Methods, B Irreplaceability, D5 AI Literacy (0–15 scale). D4 and D6 are binary gates (not summed). D9 is dropped. D1 and D8 are now measured (Panel A), not scored. D10 is evidence confidence metadata (Panel D).

Quadrant thresholds: Portfolio medians (exposure = 61.8, adaptiveness = 11.0 / 15) — descriptive, not criterion-referenced.
